Sample Prompts

Use these prompts after uploading only the files inside chatgpt-project/ into a ChatGPT Project.

1. Build the ledger

Create a value realization governance ledger from the following approved value claims. Use the runtime files. Separate benefit hypotheses from measured outcomes. Capture baseline, target, actual, measure source, measure owner, evidence confidence, realization risk, evidence gap, and scale / continue / hold / retire recommendation for human review.

2. Challenge the evidence

Review this benefits ledger as a skeptical PMO and finance partner. Do not invent missing numbers. Identify unsupported claims, missing baselines, weak sources, proxy metrics, stale evidence, unclear ownership, and finance-sensitive claims that require validation.

3. Prepare executive summary

Prepare an executive-ready benefits review summary. Separate supported, maturing, at-risk, and unsupported benefits. Identify the top decisions required, the owner actions to send to PMO Governance Operations Log, and the benefits summary to send to Executive Portfolio Review Pack Builder.

4. Create next-cycle follow-up

Create a short follow-up register for the next review cycle. Include only unresolved evidence gaps, high realization risks, missing owners, finance validation needs, and sponsor decisions. Frame all recommendations as requiring human review.
Reminder: Do not use this project to certify savings, approve continuation, accept risk, or make executive decisions.
