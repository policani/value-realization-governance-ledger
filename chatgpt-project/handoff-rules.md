# Handoff Rules

## To PMO Governance Operations Log

Send unresolved:

- owner actions;
- missing baselines;
- missing actuals;
- disputed measures;
- high realization risks;
- sponsor decisions;
- finance validation requests;
- dependency blockers;
- follow-up dates.

Do not send routine ledger rows that require no action.

## To Executive Portfolio Review Pack Builder

Send:

- benefits review summary;
- supported / maturing / at-risk / unsupported benefit counts;
- high-risk benefits requiring executive attention;
- scale / continue / hold / retire review recommendations;
- decisions required;
- evidence gaps that affect executive confidence.

## Do not pass downstream

- unsupported financial claims as certified savings;
- private source data not needed for executive review;
- raw working notes when summarized evidence is enough;
- agent recommendations framed as decisions.
