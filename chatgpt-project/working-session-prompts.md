# Working Session Prompts

## Build a ledger

Create a value realization governance ledger from these approved value claims. Normalize each record into benefit type, hypothesis, metric, baseline, target, actual, source, owner, confidence, realization risk, evidence gap, and human review route.

## Review evidence confidence

Assess the evidence confidence of this benefits ledger. Flag missing baselines, missing actuals, weak sources, proxy metrics, stale data, missing owners, and finance-sensitive claims that require validation.

## Prepare executive benefits summary

Turn this benefits ledger into an executive-ready benefits review summary. Separate supported, maturing, at-risk, and unsupported benefits. Include decisions required and handoffs to adjacent modules.

## Create follow-up actions

From this ledger, identify unresolved owner actions that should go to the PMO Governance Operations Log. Keep the action list short, specific, and owner-oriented.

## Challenge weak value claims

Challenge these benefit claims as a skeptical PMO / finance partner. Do not invent numbers. Identify weak framing, missing evidence, questionable metrics, and stronger measurement options.
