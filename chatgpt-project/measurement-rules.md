# Measurement Rules

## Measurement contract

Every benefit should answer:

1. What outcome was expected?
2. What metric proves or disproves progress?
3. What was the baseline?
4. What target was approved?
5. What actual result is available?
6. What source produced the measurement?
7. Who owns the measure?
8. How often should it be reviewed?
9. What validation is still required?

## Baseline rules

- Use the pre-change value whenever possible.
- If no baseline exists, mark it as missing and recommend reconstruction only if the human owner approves the method.
- Do not invent historical baselines.

## Target rules

- Keep approved targets separate from later forecasts.
- If the target changed, show both the original target and revised target with owner confirmation.

## Actual rules

- State the period measured.
- State the source.
- Distinguish observed result from estimated result.
- Flag self-reported actuals as lower confidence unless corroborated.

## Financial rules

Financial benefits require finance-approved methods before they become official. Until then, label them as estimates, directional indicators, or pending finance confirmation.
