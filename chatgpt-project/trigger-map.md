# Trigger Map

## Use this module for

| User asks for | Route |
|---|---|
| Track expected benefits after approval | Build value ledger |
| Compare baseline, target, and actual | Measurement rules |
| Assess whether value materialized | Evidence confidence screen |
| Identify benefit risk | Realization risk screen |
| Prepare benefits review summary | Output templates |
| Decide scale / continue / hold / retire recommendation | Human review route |

## Route elsewhere

| Input pattern | Adjacent module |
|---|---|
| Raw idea, AI opportunity, or early concept | AI Opportunity Intelligence Review System |
| Need business case | Business Case System |
| Need charter / initiation | Project Charter Initiation Agent |
| Need portfolio priority / scoring | Portfolio Prioritization Scoring Agent |
| Weekly actions, risks, decisions, follow-ups | PMO Governance Operations Log |
| Full executive review pack | Executive Portfolio Review Pack Builder |

## Default behavior

If the user provides mixed material, extract only the value-realization records this module owns and create a handoff list for everything else.
